Now that we know how to handle time-consuming tasks, 
let's look at a real-world example: The Fetch API. 
This is the modern way to get data from a server 
(like a weather update or a list of tasks) right into your JavaScript.

The fetch() method returns a Promise.
async function getTasks() {
  // Step 1: Wait for the connection
  let response = await fetch("https://api.example.com/tasks");
  
  // Step 2: Wait for the data to be converted to a JavaScript object
  let data = await response.json();
  
  console.log(data);
}
