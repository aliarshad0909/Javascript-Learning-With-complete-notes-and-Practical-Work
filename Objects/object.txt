We've used Arrays for lists (like ["Ali", "Sara"]). 
But what if you want to store a lot of different information about one specific thing?
For example, a user named Ali isn't just a name. He has an age, an email, and a login status. Instead of 3 separate variables, we use an Object.
Objects use curly braces {} and store data in key-value pairs.

Example of an Object
JavaScript
let user = {
    name: "Ali",
    age: 25,
    isLoggedIn: true
};
To get a specific piece of info, you use 
"dot notation": user.name would give you "Ali".
