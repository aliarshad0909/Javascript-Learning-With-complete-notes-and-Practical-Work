JavaScript destructuring and the spread operator (...) 
are essential features introduced in ES6 that simplify how you extract,
 copy, and merge data from arrays and objects. 

1. Destructuring
Destructuring allows you to "unpack" values from arrays or properties 
from objects directly into distinct variables. 

Object Destructuring: Extracts properties by their key names.
javascript
const user = { name: 'Alice', age: 25 };
const { name, age } = user; // name = 'Alice', age = 25
Renaming: Use key: newName to assign to a different variable name.
Default Values: Assign a value if the property doesn't exist (e.g., { name, city = 'Unknown' }).
Array Destructuring: Extracts values based on their index position.
javascript
const colors = ['red', 'green', 'blue'];
const [first, second] = colors; // first = 'red', second = 'green'
Skipping Items: Use commas to skip elements (e.g., const [first, , third] = colors). 

2. Spread Operator (...)
The spread operator "spreads" the elements of an iterable (like an array) or the properties of an object into a new context. 

Merging & Copying: Create shallow copies or combine multiple data structures into one.
javascript
const arr1 = [1, 2];
const arr2 = [...arr1, 3, 4]; // [1, 2, 3, 4]

const obj1 = { a: 1 };
const obj2 = { ...obj1, b: 2 }; // { a: 1, b: 2 }
Function Arguments: Pass an array of values as individual arguments to a function.
javascript
const nums = [5, 10, 15];
Math.max(...nums); // Equivalent to Math.max(5, 10, 15)
 

3. Combining Both (Rest Pattern)
When used on the left side of an assignment (inside destructuring), the ... syntax is called the Rest Pattern. It collects all remaining properties or elements into a new object or array. 

javascript
const [first, ...others] = [10, 20, 30, 40];
// first = 10, others = [20, 30, 40]

const { name, ...rest } = { name: 'Bob', age: 30, job: 'Dev' };
// name = 'Bob', rest = { age: 30, job: 'Dev' }

Key Differences
Feature 	Context	Typical Use Case
Destructuring	Assignment / Parameters	Pulling specific data out of a structure.
Spread	Literal / Function Call	Building new structures or copying data.
Rest	Inside Destructuring	Gathering "everything else" into a variable.
