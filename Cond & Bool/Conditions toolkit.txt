To make decisions, JavaScript uses Comparison Operators. 
These are symbols that compare two values and result in either true or false.
The Comparison Toolkit 
Symbol,Meaning,Example,Result
>          Greater than                         10 > 5                      true
<          Less than                            3 < 1                       false
>=         Greater than or equal to             60 >= 60                    true
<=         Less than or equal to                5 <= 10                     true
===        Strict equality (Exactly equal)      5 === 5                     true
!==        Not equal                           5 !== 10                     true

Real-world scenario:
Questions:
If you were writing a program for a Smart Thermostat 🌡️, 
and you wanted to check if it's "Hot," "Cold," or "Just Right," 
how many conditions (if/else if) do you think you would need to set up?
(conditions.png shows how it work in a visualized way)

Answer :
let temp = 30;
if (temp > 25) {
    console.log("It's a hot day");
}
else if (temp > 15) {
    console.log("It's a pleasant day");
}
else {
    console.log("It's a cold day");
}
